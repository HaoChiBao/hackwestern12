# CSRNet-pytorch

## Setup

```bash
# Create a virtual environment
python -m venv venv
# Activate the virtual environment (Windows)
venv\\Scripts\\activate

# Install required packages
pip install -r requirements.txt
```

## Run the model

1. **Download a pretrained model** (e.g., `model.pth`) and place it in the `weights/` directory.
2. **Start the model**:

```bash
python master.py
```
